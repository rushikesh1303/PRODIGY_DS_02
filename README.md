# PRODIGY_DS_02  

---

## **Task 2: Data Cleaning & Exploratory Data Analysis**

This repository contains **Task 02** of the **Prodigy InfoTech Data Science Internship**.

---

### **Problem Statement**

Perform data cleaning and exploratory data analysis (EDA) on a dataset (Titanic dataset). Explore relationships between variables and identify patterns and trends in the data.

---

### **Dataset**

The dataset used in this task is the **Titanic dataset**, which contains passenger details such as age, gender, class, fare, and survival status.

---

### **Work Done**

- Loaded the Titanic dataset  
- Handled missing values (**Age, Embarked, Cabin**)  
- Performed data cleaning and preprocessing  
- Analyzed survival distribution  
- Explored survival based on **gender** and **passenger class**  
- Visualized **age** and **fare** distribution  
- Identified patterns and relationships between variables
